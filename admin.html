API & Connection

Configure the AI chat assistant and your GitHub publish connection.

🔑 OpenAI API Key

Your API key is written into api/chat.js when you publish. It never appears in any public file. Get yours at platform.openai.com/api-keys

🧠 OpenAI Assistant ID — connect your custom Peques Assistant

This connects the chat widget on your website directly to your custom OpenAI Assistant. Once set, every visitor conversation runs through your Assistant — including its knowledge, files and instructions.

Find your ID at platform.openai.com/assistants — open your Assistant, copy the ID shown at the top (starts with asst_).

If left empty, the chat will use the system prompt in Chat Config instead.
⚙️ Model (fallback mode only)
Only used when no Assistant ID is set. Ignored when using your OpenAI Assistant.
🐙 GitHub — Publish Connection

Paste your GitHub Personal Access Token below. This is the only thing you need to enter — repository and branch are pre-configured. Once saved, you will not need to enter it again.

Create a token with repo scope and No expiration at github.com/settings/tokens/new →

ℹ️ How publishing works

↑ Publish to site commits two files to GitHub:

  • content.json — all text content and images
  • api/chat.js — system prompt, model, API key

Vercel detects the commit and redeploys in ~30 seconds automatically.
