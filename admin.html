Careers Page

The careers page (careers.html) is managed separately in GitHub. Use this panel for quick reference.

To edit the Careers page content (roles, form, text), open careers.html directly in GitHub and edit the HTML. The file is clearly commented — search for the section you want to update.

  1. Go to your GitHub repository
  2. Click on careers.html
  3. Click the pencil icon ✏️ to edit
  4. Search for the text you want to change (Ctrl+F in the editor)
  5. Commit the change — Vercel redeploys automatically

Formspree email routing: To change where application forms are sent, find careers@peques.co.uk in the Formspree dashboard and update the notification email there — no code change needed.

Chat Assistant

Configure the AI chat widget that appears on the landing page. Changes to the system prompt take effect on next publish.

🔑 OpenAI API Key

The API key is stored securely as a Vercel environment variable — it never appears in the code. To update it:

  1. Go to vercel.com → your project → Settings → Environment Variables
  2. Find OPENAI_API_KEY and click Edit
  3. Paste your new key (from platform.openai.com/api-keys)
  4. Save, then go to Deployments and click Redeploy
Model is set in api/chat.js and updated on publish.

ℹ️ How publishing works

When you click ↑ Publish Changes, this admin writes to two files in your GitHub repository:

  • content.json — all editable text content
  • api/chat.js — chat assistant configuration

Vercel detects the GitHub commit and redeploys automatically. The live site updates within ~30 seconds.
